# Contributing to Awesome Purple Heart Resources

First, thank you for helping grow this resource list! Every addition helps a veteran or their family find the support they need.

This guide explains how to add or update resources — no technical experience required.

---

## Ways to Contribute

- **Add a new resource** — a website, program, organization, or benefit
- **Fix a broken link** — if a URL is no longer working
- **Improve a description** — if something is outdated or unclear
- **Add a new state section** — help us grow beyond Missouri
- **Report an issue** — if you spot something wrong but aren't sure how to fix it

---

## Guidelines for Adding Resources

Before submitting a resource, please make sure it meets these standards:

- Directly benefits veterans, Purple Heart recipients, or their families
- The link is active and goes to the correct page
- It is a legitimate organization, government agency, or program
- It is free to access (or clearly labeled if there is a cost)
- No spam, advertising, or self-promotion
- No duplicate resources already on the list
